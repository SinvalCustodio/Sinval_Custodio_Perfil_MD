## ⚙️ GitHub Actions — Para Ativar os Widgets Dinâmicos

<details>
<summary>🐍 <strong>Snake Animation</strong> — Clique para ver como configurar</summary>

**1.** No repositório `SinvalCustodio`, vá em: **Settings → Actions → General → Allow all actions** ✅

**2.** Crie o arquivo `.github/workflows/snake.yml`:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"   # Roda todo dia à meia-noite
  workflow_dispatch:        # Permite rodar manualmente

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - name: Checkout
        uses: actions/checkout@v3

      - name: Generate Snake (Light + Dark)
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: SinvalCustodio
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark&color_snake=00A1E0

      - name: Deploy to output branch
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./dist
          publish_branch: output
          commit_message: "chore: update snake animation [skip ci]"
```

**3.** Execute manualmente em **Actions → Generate Snake Animation → Run workflow**

**4.** Aguarde ~2 minutos. A Snake aparece automaticamente! 🐍

</details>

<details>
<summary>🏙️ <strong>3D Contribution Graph</strong> — Clique para ver como configurar</summary>

**1.** Crie o arquivo `.github/workflows/3d-contrib.yml`:

```yaml
name: Generate 3D Contribution Graph

on:
  schedule:
    - cron: "0 18 * * *"   # Roda todo dia
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    name: Generate 3D Contribution Graph
    steps:
      - uses: actions/checkout@v3

      - uses: yoshi389111/github-profile-3d-contrib@0.7.1
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          USERNAME: SinvalCustodio
          MAX_REPOS: 10
          SETTING_JSON: |
            {
              "githubUserName": "SinvalCustodio",
              "type": "NORMAL",
              "primaryColor": "#00A1E0",
              "secondaryColor": "#032D60",
              "backgroundColor": "#1a1b27",
              "skinColor": "#ffd5b4",
              "hairColor": "#000000"
            }

      - name: Commit & push 3D graph
        run: |
          git config user.name github-actions
          git config user.email github-actions@github.com
          git add -A
          git commit -m "chore: update 3D contribution graph [skip ci]" || exit 0
          git push
```

**2.** Execute manualmente em **Actions → Generate 3D Contribution Graph → Run workflow**

</details>
